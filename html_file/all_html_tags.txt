Basic Structure of the Document:

<!DOCTYPE> - Declares the document type (HTML5)
<html> </html> - root element of an HTML document
<head> </head> - contains meta data or info about the document
<title> </title> - Displays the title for the webpage (present inside the head)
<body> </body> - contains visible content of the document 

for eg:

<!DOCTYPE>
<html>
<head>
<title> </title>
</head>
<body>
</body>
</html>

Formatting tags:

<h1> </h1> till <h6> </h6> - heading tag (varies from h1 to h6)
<p> </p> - paragraph tag 
<b> </b> or <strong> </strong> - bold or strong tag
<i> </i> or <em> </em> - italic or emphasis tag
<u> </u> - underline tag
<mark> </mark> - mark tag (to highlight any sentences)
<small> </small> - small tag (smaller text)
<del> </del> - delete tag (strike through text)
<ins> </ins> - inserted text (underlined text)
<sub> </sub> - subscript tag
<sup> </sup> - superscript tag
<br> - line break
<hr> - horizontal rule
<blockquote> </blockquote> - Quoted section
<pre> </pre> - preformatted text
<abbr> </abbr> - abbreviation tag
<cite> </cite> - cite tag
<div id="" class="" style="" title=""> </div>
<ul type=""> <li> </li> </ul> - unordered list
<ol> <li> </li> </ol> - ordered list

eg: 

<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <title>HTML Tags Example</title>
</head>
<body>

  <!-- Headings -->
  <h1>This is Heading H1</h1>
  <h2>This is Heading H2</h2>
  <h3>This is Heading H3</h3>
  <h4>This is Heading H4</h4>
  <h5>This is Heading H5</h5>
  <h6>This is Heading H6</h6>

  <!-- Paragraph -->
  <p>This is a paragraph tag that contains several inline formatting examples.</p>

  <!-- Bold / Strong -->
  <b>This is bold text using &lt;b&gt;</b><br>
  <strong>This is important text using &lt;strong&gt;</strong><br>

  <!-- Italic / Em -->
  <i>This is italic text using &lt;i&gt;</i><br>
  <em>This is emphasized text using &lt;em&gt;</em><br>

  <!-- Underline -->
  <u>This text is underlined using &lt;u&gt;</u><br>

  <!-- Mark (Highlight) -->
  <mark>This sentence is highlighted using &lt;mark&gt;</mark><br>

  <!-- Small Text -->
  <small>This is small text using &lt;small&gt;</small><br>

  <!-- Deleted / Inserted Text -->
  <del>This is deleted text using &lt;del&gt;</del><br>
  <ins>This is inserted text using &lt;ins&gt;</ins><br>

  <!-- Subscript / Superscript -->
  H<sub>2</sub>O (subscript for water)<br>
  E = mc<sup>2</sup> (superscript for energy formula)<br>

  <!-- Line Break -->
  This is one line.<br>
  This is after a line break.<br><br>

  <!-- Horizontal Rule -->
  <hr>

  <!-- Blockquote -->
  <blockquote>
    “The best way to get started is to quit talking and begin doing.” – Walt Disney
  </blockquote>

  <!-- Preformatted Text -->
  <pre>
Line 1
    Line 2 (indented)
Line 3
  </pre>

  <!-- Abbreviation -->
  <p>The <abbr title="World Health Organization">WHO</abbr> was established in 1948.</p>

  <!-- Cite -->
  <p><cite>The Ramayana</cite> is a famous ancient Indian epic.</p>

  <!-- Div with attributes -->
  <div id="container" class="box highlight" style="border: 2px solid black; padding: 10px;" title="Main Div">
    This is a styled <b>&lt;div&gt;</b> container with id, class, style, and title attributes.
  </div>

  <br><br>

  <!-- Unordered List -->
  <ul type="square">
    <li>HTML</li>
    <li>CSS</li>
    <li>JavaScript</li>
  </ul>

  <!-- Ordered List -->
  <ol>
    <li>Wake up</li>
    <li>Brush teeth</li>
    <li>Eat breakfast</li>
  </ol>

</body>
</html>

anchor tags:

<a href="">displays content</a> -  basic syntax

types of links in anchor tags:

1. External link : <a href="other webpage link" target="_blank">display content</a> - Links to another website (Opens link in new browser tab)
2. internal link : <a href="about.html">About Us</a> - Links to a page within the same website
3. I-page link : <a href="#contact">Go to Contact</a> (use with <div id="contact">) - Jumps to a section within the same page
4. Email link : <a href="@gmail.com">name</a> 
5. phone link : <a href="+91 7789654123>phone numnber</a>
6. Download link : <a href="file.pdf" download>download pdf</a>

link tags : 

1. link the external css - <link rel="stylesheet" href="style.css">
2. to display the icon - <link rel="icon" href="favicon.ico" type="image/x-icon">
